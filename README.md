ClipItBro - 高精度動画変換ツール
====================================

## 概要
ClipItBroは、プロフェッショナル品質の動画変換を簡単に実現するPyQt5ベースのGUIアプリケーションです。
FFmpegの2passエンコーディングシステムを活用し、目標ファイルサイズに対する高精度な変換を提供します。

## 主要機能

### 🎯 高精度2passエンコーディング
- **2pass方式**: 目標ファイルサイズを正確に達成する高精度変換
- **CRF方式**: 品質重視の従来型エンコーディング
- **自動1pass分析**: ドラッグ&ドロップ時に最適ビットレート自動計算
- **リアルタイム進行表示**: デュアルプログレスバーで1pass/2passの進行状況を可視化

### 🎨 洗練されたユーザーインターフェース
- **ダーク/ライトテーマ**: システム全体の統一されたテーマ切り替え
- **タイトルバー連動**: アプリテーマに合わせてタイトルバーの色も自動変更
- **直感的操作**: ドラッグ&ドロップによる簡単ファイル読み込み
- **視覚的フィードバック**: 矢印アイコンで処理フローを明確化

### 📊 詳細な動画分析
- **完全な動画情報表示**: 解像度、フレームレート、長さ、ビットレート、ファイルサイズ
- **ファイルサイズ推定**: 変換後のサイズとファイルサイズ変化率を事前計算
- **インテリジェント品質制御**: CRF値とVideo Filter強度の細かな調整

### 🛡️ 堅牢なエラーハンドリング
- **FFmpeg検出システム**: 依存関係の自動チェックと適切なエラー表示
- **完全機能無効化**: FFmpeg未検出時は全操作を安全に無効化
- **視覚的状態表示**: スライダーやボタンの無効化で操作不可状態を明確化

## 使用方法

### 基本操作
1. **アプリケーション起動**: main.pyを実行
2. **ファイル読み込み**: 動画ファイルをテキストエリアにドラッグ&ドロップ
3. **自動分析**: 1pass分析が自動実行され、最適設定が計算されます
4. **変換方式選択**: 
   - **2pass**: ファイルサイズスライダーで目標サイズを設定
   - **CRF**: CRFスライダーで品質レベルを調整
5. **変換実行**: 「変換実行」ボタンをクリック
6. **進行確認**: デュアルプログレスバーで1pass/2passの進行状況を監視

### 高度な機能
- **テーマ切り替え**: メニューバー「表示」→「テーマ」から選択
- **変換方式切り替え**: 2passボタンでワンクリック切り替え
- **Video Filter調整**: vfスライダーで画質向上フィルターの強度調整

## インターフェース説明

### メインコントロール
- **ファイルサイズスライダー**: 目標ファイルサイズ設定（2pass方式）
- **CRFスライダー**: 品質レベル設定（CRF方式）
- **VFスライダー**: Video Filter強度調整
- **2pass/CRFボタン**: エンコーディング方式切り替え

### 進行状況表示
- **1pass目プログレスバー**: 分析フェーズの進行状況
- **矢印インジケーター**: 処理フローの視覚化
- **2pass目プログレスバー**: 最終変換フェーズの進行状況

### メニュー機能
- **ファイル**: アプリケーション終了
- **表示**: テーマ切り替え（ライト/ダーク）
- **ヘルプ**: バージョン情報とAboutページ

## カスタマイズ

### アプリケーションアイコン
以下のいずれかのファイルを配置：
- icon/app.ico
- icon/app.png  
- app.ico（ルートフォルダ）
- app.png（ルートフォルダ）

### 制作者ロゴ（Aboutページ用）
以下のいずれかのファイルを配置：
- icon/logo.png, logo.jpg, logo.ico
- logo.png, logo.jpg, logo.ico（ルートフォルダ）

推奨サイズ：100x100ピクセル

### 変換完了通知アイコン（ランダム表示）
```
icon/completion/
├── general/        # 汎用画像（優先度：低）
├── seasonal/       # 季節画像（優先度：中）  
├── special/        # 特別画像（優先度：高）
└── premium/        # プレミアム画像（優先度：最高）
```

対応形式：.png, .jpg, .jpeg, .bmp, .gif, .svg
注意：ファイル名に"thumb"が含まれるファイルは除外

## システム要件
- **OS**: Windows 10/11（タイトルバーテーマ対応）
- **Python**: 3.x
- **依存ライブラリ**: PyQt5
- **FFmpeg**: bin/フォルダ内に配置必須

## 技術仕様

- **エラー復旧**: 変換失敗時の適切なクリーンアップ
